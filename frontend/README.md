# frontend

The React/TypeScript client for the CRM. Scaffolded with Vite.

See the repo root's `CLAUDE.md` and `docs/` for the actual rules this code follows — in particular [`docs/frontend-design-system.md`](../docs/frontend-design-system.md) (UI library/theme decisions), [`docs/frontend-coding-standards.md`](../docs/frontend-coding-standards.md), and [`docs/testing-strategy.md`](../docs/testing-strategy.md).

## Commands

```
npm run dev       # dev server
npm test          # Vitest
npm run lint      # ESLint
npm run format    # Prettier --write
npm run build     # type-check + production build
```
